Canvas Guitar

Listen to the calming sounds of nature as you strum your guitar.
Change your environment with the dropdown menu below.

    Instructions

  • Hold the left mouse button and
    mouse over a string to tug on it.

  • Pull it far enough or release the
    left mouse button and it will be plucked.

  • Keyboard Controls

    • Left Arrow - Pluck 1st string

    • Down Arrow - Pluck 2nd string

    • Right Arrow - Pluck 3rd string

    • Numpad 4 - Pluck 4th string

    • Numpad 5 - Pluck 5th string

    • Numpad 6 - Pluck 6th string

    • Spacebar - Mute

    • Hold 1, 2, 3, 4, 5, 6, 7, 8, 9, 0, -, or =
      to press fret numbers 1-12
