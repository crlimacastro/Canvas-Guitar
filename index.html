Canvas Guitar

Listen to the calming sounds of nature as you strum your guitar. Change your surroundings at the click of a button.

Instructions

  • Hold the left mouse button and mouse over a string to tug on it.

  • Pull it far enough or release the left mouse button and it will be plucked.

    • Keyboard Controls

  • Left Arrow - Pluck 1st string

  • Down Arrow - Pluck 2nd string

  • Right Arrow - Pluck 3rd string

  • Numpad 4 - Pluck 4th string

  • Numpad 5 - Pluck 5th string

  • Numpad 6 - Pluck 6th string
